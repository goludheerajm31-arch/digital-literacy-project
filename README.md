##  Project Overview

This repository contains my **Digital Literacy Portfolio Project**, completed as part of the Digital Literacy course at VIT Bhopal University.  

In this project, I worked as a **Student Digital Ambassador**, learning how to use digital tools responsibly, build a professional online presence, communicate effectively online, and understand cyber safety practices.

The project consists of five modules covering digital awareness, professional portfolios, collaboration platforms, email etiquette, and cybercrime prevention.
